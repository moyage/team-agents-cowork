# V0.8.8 Iteration Plan: Documentation Usability & Workflow Deep Dive

## Objective
Establish the framework for the documentation overhaul focusing on the 18 Built-in Workflows, Custom Workflow Authoring, and Repository Sinicization.

## Phases

### Phase 1: Repo Sinicization (中文)
- Establish Chinese documentation as the absolute primary focus.
- Align and update the primary READMEs to reflect this priority.
- Ensure all foundational concepts are clearly documented in Chinese.

### Phase 2: Built-in Workflow Deep Dive
- Audit and read the 18 YAML workflow templates.
- Create visual Mermaid diagrams for each of the 18 built-in workflows.
- Document inputs, outputs, and execution paths for each template.

### Phase 3: Custom Workflow Methodology
- Document the framework for authoring custom workflows.
- Explain trigger conditions and how they are evaluated.
- Outline boundary decisions (when to use built-in vs. custom workflows).
- Provide practical examples and tutorials for custom workflow creation.
