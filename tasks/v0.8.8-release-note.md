# Release v0.8.8: Docs Usability & Workflow Deep Dive

## Overview
Iteration V0.8.8 drastically improves the framework's onboarding experience and operational clarity by shifting the primary documentation focus to comprehensive Chinese guides. It provides Archon-level depth into the 18 built-in workflows and custom workflow authoring mechanics.

## Key Additions
- **18 Built-in Workflows Demystified**: Fully mapped all 18 YAML workflows (`idea-to-pr`, `piv-loop`, `architect`, etc.) in `documentation/ZH/reference/built-in-workflows.md`. Each entry now contains Use Cases, Trigger Conditions, Selection Strategy, Mermaid DAG Flowcharts, and Execution Examples.
- **Custom Workflow Authoring Matrix**: Added a dedicated guide (`documentation/ZH/guides/authoring-workflows.md`) defining the boundary decision matrix (When to use built-in vs custom). It explicitly details how to inject `delegation_mode: blackbox | orchestrated` and provides a robust real-world CI/CD multi-agent example.
- **Chinese-First Pivot**: Restructured the root `README.md` and `README_zh-CN.md` to point users first to the meticulously crafted Chinese documentation center for the lowest cognitive load.
