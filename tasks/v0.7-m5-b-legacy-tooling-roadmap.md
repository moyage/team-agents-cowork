# V0.7-M5-B: Legacy Tooling Deprecation Roadmap

## 1. Objective
Provide a structured deprecation path for the legacy Python tooling (`scripts/*.py`). As the Node.js MCP server assumes full responsibility for phase gating and workflow state validation, these duplicated tools must be safely phased out without breaking existing pipelines.

## 2. Tooling Inventory & Classification
The `scripts/` directory contains tools built for V0.3/V0.4. They are classified into three actionable categories:

### A. Deprecate (Targeted for Removal in V0.8)
These scripts replicate logic that is now 100% covered by the `src/mcp-server/` APIs.
- **`scripts/doctor.py`**: Previously used to verify protocol states. Replaced by `get_dispatch_state`.
- **`scripts/verify_protocol.py`**: Replaced by `verify_execution_compliance` MCP tool.
- **`scripts/validate_workflow_state.py`**: Replaced by `get_iteration_state`.
- **`scripts/build_demand_pool.py`**: Obsolete. Task decomposition is now dynamically managed via Coordination Agents generating JSON execution contracts.

### B. Retain (Utility & Bootstrap)
These scripts perform low-level OS tasks or CI checks that fall outside the typical L2/L3 MCP communication loop.
- **`scripts/verify_integrity.py` & `scripts/update_integrity_manifest.py`**: Retain for anti-tamper CI pipelines.
- **`scripts/render_adapters.py` (if any)**: Retain temporarily as a fallback until the NPM CLI (`agent-protocol-init`) is universally adopted across all team configurations.

### C. Remove (Immediate / Next Phase)
- **`scripts/sync_to_target.py`**: This was already explicitly deprecated in V0.6-M4 when `setup.sh` was migrated to call `node ./bin/agent-protocol-init`.

## 3. Deprecation Rules & Criteria
1. **No Silent Breakage:** Before physically deleting a script from category A, a deprecation warning must be injected into its execution output for one full iteration (V0.7).
2. **MCP Parity Check:** A script cannot be moved to category C unless the equivalent MCP tool has been integration-tested in both Personal and Team Modes.

## 4. Execution Plan (Deferred)
No scripts are deleted in M5-B. The actual deletion contract will be issued in V0.8, following the deprecation warning phase.
