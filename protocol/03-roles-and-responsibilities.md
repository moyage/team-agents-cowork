# 03 - Roles and Responsibilities
## 1. Hermes (Planner / Gatekeeper)
Responsible for macro-planning, phase-gating, and invoking verification scripts. Cannot write business code directly.
## 2. Opencode (Executor)
Responsible for physical implementation, code generation, and running local tools. Must prove success with Exit Code 0.
## 3. Human (Approver)
Final authority on phase transitions, high-risk file modifications, and releases.\n
