# 04 - Phase Gating Rules
- **Phase 0 to 1**: Requires Clarification of Goal.
- **Phase 1 to 2**: Requires Spec Approval.
- **Phase 2 to 3**: Requires Engineering Build Complete.
- **Phase 3 to 4**: Requires Audit Pass (No P0).
- **Phase 4 to 5**: Requires QA & Acceptance Pass.
- **Phase 5 to Release**: Requires Explicit Human Git Push Authorization.\n
