# 17 Built-in Workflows

Our framework ships with 17 high-frequency DAG workflows.

| ID | Description |
|---|---|
| `assist` | General Q&A, debugging, exploration |
| `fix-github-issue` | Classify issue → plan → implement → validate → PR → smart review |
| `idea-to-pr` | Feature idea → plan → implement → validate → PR → 5 parallel reviews |
| `plan-to-pr` | Execute existing plan → implement → validate → PR → review |
| `issue-review-full` | Comprehensive multi-agent review pipeline for GitHub issues |
| `smart-pr-review` | Classify PR complexity → run targeted review agents |
| `comprehensive-pr-review` | Multi-agent PR review (5 parallel reviewers) |
| `create-issue` | Classify problem → gather context → investigate → create GitHub issue |
| `validate-pr` | Thorough PR validation testing both main and feature branches |
| `resolve-conflicts` | Detect merge conflicts → analyze both sides → resolve |
| `feature-development` | Implement feature from plan → validate → create PR |
| `architect` | Architectural sweep, complexity reduction, codebase health |
| `refactor-safely` | Safe refactoring with type-check hooks |
| `ralph-dag` | PRD implementation loop |
| `remotion-generate` | Generate Remotion video compositions with AI |
| `test-loop-dag` | Loop node test workflow |
| `piv-loop` | Guided Plan-Implement-Validate loop |
