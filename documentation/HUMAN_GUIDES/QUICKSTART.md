# Quickstart Guide

Welcome to the Team Agents Cowork framework! 

## Installation
The framework's configuration adapters can be installed locally into your project:
```bash
# This command will be available in V0.6-M4
# agent-protocol-init
```

## Running Your First Agent
Once configured, you can launch your agent (e.g. OpenCode, Cursor) and ask it to read the `workflow/dispatch.json` file.

*For more details on how agents interpret our rules, see the [AI Guides](../AI_GUIDES/SYSTEM_PROMPT.md).*
