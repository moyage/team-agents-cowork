# Architecture & L0-L6 Paradigm

The Team Agents Cowork framework is based on a rigid, layered architecture (L0-L6) that defines constraints from high-level product strategy down to code implementation and evaluation.

- **L0: Product Vision**
- **L1: Feature Blueprint**
- **L2: Meta-Workflow (The Router)** — Controlled by `dispatch.json`
- **L3: Protocol Schema & Components** — Verified by the MCP Server
- **L4: Implementation**
- **L5: Integration & E2E Verification**
- **L6: Evaluator & Gating** — Executed by independent SubAgents

*Note: The canonical definitions of these layers previously resided in the `protocol/` directory and are actively being merged into this unified structure.*
