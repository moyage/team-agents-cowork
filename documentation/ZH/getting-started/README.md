# 快速开始 (Getting Started)

欢迎来到 Team Agents Cowork 的快速开始指南。本章节旨在帮助您快速完成框架的基础环境搭建，并通过一系列循序渐进的操作，建立并运行您的第一个多智能体协作空间。

通过本章节的学习，您将能够从零开始体验 Team Agents Cowork 强大的多智能体编排能力，了解系统的基本运作方式。
