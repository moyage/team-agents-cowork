# Agent Collab Kit (Team Agent Collaboration Protocol)

这是一个自举式、可版本化、可安装的团队 Agent 协作协议工程主仓库。

## 目录索引
- `protocol/`：团队协作规范与工作流（Canonical Source）。
- `templates/`：任务流转模板 (SPEC, TASK_BREAKDOWN, VERIFY, PR 等)。
- `adapters/`：针对不同个人 Agent/IDE 环境的动态适配器。
- `scripts/`：用于安装、渲染、检查和升级的自动化脚本。

## 快速开始
```bash
make install
```
